Business rules

All renderings and views will be in french.

In Enum types CHOOSE will not be an option.

Supress value when field is hiden.

The rules behind the different fields

Date Of Transaction

JSON Format yyyy-MM-dd

Java type : Date

Java variable name : dateOfTransaction

MANDATORY

Type Of Transaction

Java type : Enum

Java variable name : TypeOfTransaction

MANDATORY

If the type of transaction is set to 'CHECK', then the field 'checkNumber' is mandatory.

If the type of transaction is set to 'SALARY', then the picklist of the field 'themeGeneral' is [Muriel, Patrick].

If the type of transaction is not set to 'SALARY', then the picklist of the field 'themeGeneral' does not contain [Muriel, Patrick].

Check Number

Java type : String

Java variable name : checkNumber

MANDATORY : Only when the field 'TypeOfTransaction' is set to 'CHECK'

UNIQUE

pattern : length superior to 7 digit, maximum 10 digits, composed only of digits.

Name

Java type : String

Java variable name : name

MANDATORY

Amount

Java type : Float

Java variable name : amount

MANDATORY

pattern : maximum length 10 digits, composed only of digits, must contain '.'.

Theme General

Java type : Enum

Java variable name : themeGeneral

MANDATORY

If the field 'TypeOfTransaction' is set to 'SALARY', the picklist only contains [Muriel, Patrick].

If the field 'TypeOfTransaction' is not set to 'SALARY', the picklist doese not contain [Muriel, Patrick].

If theme general is set to 'PRESENT', the field 'beneficiary' is mandatory, but the 'ThemeSub' is not.

If theme general is set to [CALUIRE_ET_CUIRE, MEYZIEU, HEALTH, TCL, CLOTHING, INCOME_TAXE], the field 'ThemeSub' is mandatory.

If theme general is set to [HEALTH, TCL, CLOTHING, , COMPUTER], the picklist of the field 'ThemSub' contains [Muriel, Patrick, Both, HouseHold]

If theme general is set to 'CALUIRE_ET_CUIRE', the picklist of the field 'ThemSub' contains [INSURANCE, RENT, COSTS, CONSTRUCTION_WORK, WATER, GASS, ELECTRICITY, PROPERTY_TAXE, BOILER].

If theme general is set to 'MEYZIEU', the picklist of the field 'ThemSub' contains [CHIMNEY_SWEEPING, HOME_APPLIANCE, FURNITURE, FIRE_WOOD, INSURANCE, LOAN, COSTS, HOUSING_TAXE, CONSTRUCTION_WORK, WATER, GASS, ELECTRICITY, PROPERTY_TAXE, BOILER].

Beneficiary

Java type : String

Java variable name : beneficiary

MANDATORY : Only when the field 'themeGeneral' is set to 'PRESENT'

Theme Sub

Java type : Enum

Java variable name : themeSub

MANDATORY : Only when the field 'themeGeneral' is set to [CALUIRE_ET_CUIRE, MEYZIEU, HEALTH, TCL, CLOTHING, INCOME_TAXE, COMPUTER]

If theme general is set to [CALUIRE_ET_CUIRE, MEYZIEU, HEALTH, TCL, CLOTHING, INCOME_TAXE], the field 'ThemeSub' is mandatory.

If theme general is set to [HEALTH, TCL, CLOTHING, INCOME_TAXE, COMPUTER], the picklist of the field 'ThemSub' contains [Muriel, Patrick, Both, HouseHold]

If theme general is set to 'CALUIRE_ET_CUIRE', the picklist of the field 'ThemSub' contains [INSURANCE, RENT, COSTS, CONSTRUCTION_WORK, WATER, GASS, ELECTRICITY, PROPERTY_TAXE, BOILER].

If theme general is set to 'MEYZIEU', the picklist of the field 'ThemSub' contains [CHIMNEY_SWEEPING, HOME_APPLIANCE, FURNITURE, FIRE_WOOD, INSURANCE, LOAN, COSTS, HOUSING_TAXE, CONSTRUCTION_WORK, WATER, GASS, ELECTRICITY, PROPERTY_TAXE, BOILER].

Tenant

Java type : String

Java variable name : tenant

MANDATORY : Only if the field ThemSub is set to 'RENT'.

Author

Java type : Enum

Java variable name : author

MANDATORY : Only the field 'themeSub' is set to [CHIMNEY_SWEEPING, HOME_APPLIANCE, FURNITURE, FIRE_WOOD, INSURANCE, LOAN, COSTS, HOUSING_TAXE, CONSTRUCTION_WORK, WATER, GASS, ELECTRICITY, PROPERTY_TAXE, BOILER].
